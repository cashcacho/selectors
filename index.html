This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Here's what it looks like:

Inside my-div
class

classes can be applied to several items on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
                background-color:orange;
            }
}

compound

Compund selectors allow us to use more that one selector for a set of rules. The selectors must be seperated by commas

Here's how the CSS is applied to the element:

li.my-list{div#new-div, li.new-list{
                background-color:blue;
}

compound

The child selector allows us to access the elements that are immediately inside the current element.

Here's how the CSS is applied to the element:

ul.my-kids > li.new-list{
                background-color:green;
}
